snmp_set_enum_print -- Return all values that are enums with their enum value
instead of the raw integer

  void snmp_set_enum_print(int enum_print)~

This function is currently not documented; only its argument list is
available.

|snmp_set_enum_print| is only available when using the UCD SNMP library. This
function is not available when using the Windows SNMP library.
