###安装ycm的步骤 for c/c++ style user
####ubuntu
sudo apt install build-essential cmake python-dev
####centos
sudo yum isntall clang cmake28

如果不使用c/c++的语义提示，就直接执行ycm项目目录的./install.sh
如果要使用语义提示需要安装clang
sudo apt-get install clang
centos must install by repel repo
如果之前就安装过了,可以用locate libclang.so查找libclang.so的位置，比如结果是/usr/lib64/llvm/libclang.so

切换到ycm的项目目录，然后执行下面的语句
./install.sh --clang-completer
或者
cmake28 -G "Unix Makefiles" . -DPATH_TO_LLVM_ROOT=/usr/lib64/llvm ~/.vim/bundle/YouCompleteMe/third_party/ycmd/cpp
make ycm_support_libs
就可以安装成功ycm了
[warning]有可能YCM在我的安装包没人没有装，你可以先取消_vimrc.bundles文件中对YCM的注释


###vba文件安装方法
如果有vba的vim插件,需要用vimball方式安装
用vim打开vba格式的文件，输入
:so %
即可安装，然后:q退出。
删除插件也很方便，直接在vim里输入
:RmVimball 插件名

比如我其中一个插件manviewpage.vba就是用这个方法(我已经默认安装)
